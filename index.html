⚙️ Guía de configuración (solo tú la ves)

  1. Crea un Google Sheet con las hojas: Partidos, Participantes, Quinielas, Noticias
  2. En el Sheet: Archivo → Compartir → Publicar en la web → formato CSV
  3. Copia la URL base y ponla en config.js como SHEET_ID
  4. Sube los 3 archivos a GitHub y activa GitHub Pages
  5. Comparte la URL con los participantes

Lee el archivo INSTRUCCIONES_SETUP.md incluido para la guía completa paso a paso.
